# 👓 Glasses Detection App — Computer Vision with Python

A Python-based computer vision application that detects whether a person is wearing glasses or not, using image processing and machine learning techniques.

---

## 🚀 Demo

> Upload a photo → The app analyzes the face → Tells you: **Glasses ✅ or No Glasses ❌**

---

## 🛠️ Technologies Used

| Technology | Purpose |
|------------|---------|
| Python | Core programming language |
| OpenCV | Image processing & facial detection |
| NumPy | Data handling & processing |
| Machine Learning | Image classification & model training |

---

## 📋 Features

- 🔍 Real-time face detection
- 👓 Glasses / No Glasses classification
- 📸 Works with image inputs
- ⚡ Fast and lightweight

## 🧠 How It Works

1. The app captures or loads an image
2. OpenCV detects the face in the image
3. The region around the eyes is extracted
4. The ML model classifies whether glasses are present
5. The result is displayed to the user
